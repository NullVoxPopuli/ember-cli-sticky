# Ember-cli-sticky

Ember addon to integrate sticky.js

## Installation

* `ember install ember-cli-sticky`

## Running

* `ember server`
* Visit your app at http://localhost:4200.

## Usage

```js
{{#sticky-container}}
<nav ... />
{{/sticky-container}}
```

## Running Tests

* `ember test`
* `ember test --server`

## Building

* `ember build`

For more information on using ember-cli, visit [http://www.ember-cli.com/](http://www.ember-cli.com/).
